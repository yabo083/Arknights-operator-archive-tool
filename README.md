# Temporal Void - 明日方舟干员档案查询工具

一个用于统计和展示明日方舟未招募干员的可视化工具。

## 功能特点

- 🎯 3D螺旋展示未招募干员
- 📊 统计数据可视化
- 🎵 背景音乐播放器
- 📸 生成分享报告
- 💾 本地数据存储

## 在线访问

访问 GitHub Pages 部署的网站：[点击这里](https://yabo083.github.io/Arknights-operator-archive-tool/formal/missing_ops.html)

## 本地使用

1. 克隆仓库
2. 直接打开 `formal/missing_ops.html` 即可使用
